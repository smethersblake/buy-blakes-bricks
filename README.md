# Buy Blake's Bricks
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
## Description
An eCommerce website to sell and purchase individual lego bricks. 

## Technologies Used
- MongoDB
- Mongoose
- Express.js
- React
- Node.js
- TailwindCSS
- Apollo GraphQL
- JWT
- bcrypt
- Jest
